CheatDevice v2.0 ALPHA for GTA Vice City Stories
By Edison Carter


To install:

 copy cheatdevice.prx X:\dh\kd

  (replace X: with the drive letter of your PSP)

 edit this file:

  X:\dh\271\flash0\kd\pspbtcnf_game.txt

   (replace 271 with any other emulated firmware versions you want to use, such as 260, 250, 200)

 and add the line

  ms0:/dh/kd/cheatdevice.prx

 after the line

  ms0:/dh/kd/devhook.prx
