CheatDevice v1.0d PRX for GTA:Liberty City Stories
By Edison Carter

This is the PRX version for use with DevHook.

Game disk versions:

 original US version         -- works
 original UK/Euro version    -- works
 new patched US version      -- works
 new patched US/Euro version -- not working


To install:

 copy cheatdevice.prx X:\dh\kd

  (replace X: with the drive letter of your PSP)

 edit this file:

  X:\dh\271\flash0\kd\pspbtcnf_game.txt

   (replace 271 with any other emulated firmware versions you want to use, such as 260, 250, 200)

 and add the line

  ms0:/dh/kd/cheatdevice.prx

 after the line

  ms0:/dh/kd/devhook.prx
